# GoQuest Sprint 3

Assinatura: Tehkne Solutions

## Objetivo

Criar uma camada medieval para o GoQuest sem alterar as regras do Go.

## Entregas

- Tipos de personagem.
- Gerador de personagens.
- Motor preparado para receber personagem por jogada.
- Celulas do tabuleiro preparadas para guardar personagem.
- Textos das missoes com narrativa de campanha.
- Visual inicial de unidade no tabuleiro.
- Banco inicial com 10 puzzles.
- Painel de personagem selecionado.
- Painel de codex de papeis.

## Proxima etapa

- Ligar o painel de personagem na tela principal.
- Criar modo livre.
- Criar debug visual de liberdades e grupos.
