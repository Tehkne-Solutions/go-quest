GoQuest Sprint 6.1 — Fantasy Textured Pieces
Tehkné Solutions

Este overlay atualiza as peças do tabuleiro 3D para um visual de fantasia mais rico, baseado na arte das miniaturas heroicas.

O que entra:
- novos assets em public/assets/goquest/pieces-textured/
- Piece3D refeito para usar miniaturas texturizadas em 2 planos cruzados (impostor 3D)
- pedestal 3D por classe preservado
- seleção e formação continuam funcionando

Como aplicar:
1. extraia o zip na raiz do repo
2. copie o conteúdo por cima do projeto
3. rode:
   npm run build
   npm run dev
